# 🏏 IPL 2026 Analytics Dashboard

An interactive Power BI dashboard built to analyze IPL 2026 match data and uncover insights into team, batting, bowling, and venue performances.

## 📊 Dashboard Features

- Tournament Overview
- Batting Performance Analysis
- Bowling Performance Analysis
- Team-wise Insights
- Venue-wise Statistics
- Interactive Filters and Navigation

---

## 🛠️ Technologies Used

- Power BI
- DAX
- Data Modeling
- Data Visualization
---

## 🎯 Key Insights

- Total Runs Analysis
- Total Wickets Analysis
- Orange Cap & Purple Cap Tracking
- Strike Rate Analysis
- Boundary Distribution
- Economy Rate Analysis
- Venue Performance Insights
